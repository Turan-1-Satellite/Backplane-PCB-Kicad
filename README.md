# About
This repository contains the KiCad 9.0 project files for the TURAN-1 Backplane PCB.

# Requirements
- KiCad 9.0 or later
- Proper configuration of the "KIPRJMOD" env variable

# First Set up
All required custom libraries and 3D models are included in this repository.
After cloning the repository, add an env variable named "KIPRJMOD" to KiCad env variables section. For the value add a path to your folder where project is located (E.g. "/my-computer/Turan1/BPB/PCB-Kicad").

# Open the Project

Open the BP-PCB-Kicad.kicad_pro file from the project root directory.

If the environment variable is configured correctly, all custom libraries and 3D models will load automatically.
