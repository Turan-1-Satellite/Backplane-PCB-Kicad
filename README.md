# Turan-1 Backplane PCB in KiCad
## About
This repository contains the KiCad 9.0 project files for the TURAN-1 Backplane PCB.

## Requirements
- KiCad 9.0 or later

## First Set up
All required custom libraries and 3D models are included as a git submodule. To clone the repository together with custom libraries, you need to recursively clone it:
```
git clone --recurse-submodules https://github.com/Turan-1-Satellite/Backplane-PCB-Kicad.gi
```

## Open the Project
Open the BP-PCB-Kicad.kicad_pro file from the project root directory.
